Estrutura básica do código HTML

(Voltar)

Inicialmente, você deve construir uma pasta para armazenar o seu projeto. Então, você deve abrir o Visual Studio Code, ir em Arquivo clicar em abrir pasta e aí selecionar a pasta que você deseja armazenar o seu código. Para início, vai em Novo Arquivo e coloca como nome Index.html, não é obrigatório, podem é o nome padrão utilizado que determina o início do código.

Introduzindo o código escreva entre os sinai maior que e menor que, as letras "html". Todas as tags, ficam entre esses dois símbolos. O código consiste em "head" e "body". No "head" ficará a parte que é pré carregada antes de mostrar para o usuário, como a tag "title" que dará o nome da páginam já no "body", ficaram as informações que são exibidas para o usuário.

Falando sobre tags

(Voltar)

Para a construção do código existirão diversas tags, como a tag "i", que deixa o testo em itálico. Para usa-lá, é necessário apenas colocar o "i" entre os sinais de

menor e maior que, e colocar o trecho que deseja alterar, entre a tag. Há tags que não precisam de "fechamento", essas atuam sozinhas como um campo, um exemplo é a tag "input". Um elemento presente nas tags são os atribultos eles estarão presentes dentro dos sinais de maior que e menor que, juntamente as letras da tag, os atribultos servem para colocar o que as tags definem de modo diferente no navegador.

Atributos das tags

(Voltar)

Existem diversos tipos de atribultos, sendo eles exclusivos de uma tag, ou globais, que serão aceitam por qualquer tag. Um exemplo de um atributo específico, é o "src", que contido na tag "img", permite a introdução de imagens no seu código. Outro atributo é o "width", que também é utilizado na tag "img", ele permite alterar a largura da imagem no site. Por exemplo a tag "id", ela no Java Script, serve para identificar qual elemento você deseja alterar, e a tag style, esta é voltada para o CSS3 e ela serve para você atribuir comandos CSS

No seu código html. Outro exemplo de tag utilizada no CSS3, é a tag "class", que permite padronizar textos.

Textos

(Voltar)

Texto é basicamento tudo que é lido pelo usuário, é a tipografia. Os títulos dos textos são determinados pelas tags "h", que vão do "h1" ao "h6". Esses "h" são divididos em níveis, como título, subtítulo, etc. Já para delimitar o conteúdo dos textos é utilizado a tag "p", ela delimitará o texto. A cada linha, deve haver uma tag "p", no final para delimitar a linha do seu texto Para construir uma citação ou observação durante o seu texto, há uma tag chamada "blockquote", ela abrirá um espaço para que vocẽ possa construir sua citação ou observação.

Listas Ordenadas e Não Ordenadas

(Voltar)

A construção de listas em HTML pode ser feita de dois modos, listas ordenadas e não ordenadas. As listas ordenadas são dadas pela tag "ol", em seguida vá para próxima linha e use a tag "li", essa tag será responsável pelo conteúdo inserido em cada linha da sua lista, ao final fechar a tag "ol".

A lista não ordenada segue o mesmo padrão, porém, ao invés de utilizar a tag "ol", utilizasse a tag "ul".

Links

(Voltar)

Para usar os colocar links é necessário a tag "a" juntamente ao atributo "href", que é específico dessa tag, ela atuará como ancora direcionando o usuário para o endereço indicado. Para a página ser aperta em outra aba pode-se usar outro atributo o "target=_blank". Outro atributo utilizado é o "title", que adiciona um balão quando o cursor do mouse estiver em cima do link.
